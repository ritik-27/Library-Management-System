### **Resume Bullet Points (Choose 3-5 most relevant):**

#### Technical Implementation
- **Architected and developed a full-stack Library Management System** using MERN stack (MongoDB, Express.js, React 18, Node.js), implementing RESTful APIs with 10+ endpoints for book inventory and user management operations

- **Built responsive React frontend** with Material-UI components, React Router v6 for client-side routing, and Context API for global state management, achieving seamless user experience across devices

- **Designed and implemented MongoDB database schema** using Mongoose ODM with optimized data models for books, users, and borrowing history, supporting efficient queries for real-time inventory tracking

- **Implemented secure user authentication system** using Express sessions with cookie-based session management, role-based access control (Admin/User), and protected API routes

- **Developed interactive data visualization dashboards** using Highcharts library to display historical price and quantity trends, enabling data-driven inventory decisions

#### Feature Development
- **Engineered complete book inventory management system** with CRUD operations, ISBN-based tracking, category classification, and real-time availability status updates

- **Built book borrowing/returning workflow** with validation logic, preventing duplicate borrows, tracking borrowed books per user, and maintaining accurate inventory counts

- **Created role-based admin panel** with protected routes for book addition, editing, and deletion, implementing admin-only access using Higher-Order Components (HOCs)

- **Implemented client-side form validation** with real-time error feedback, input sanitization, and user-friendly error notifications using React Notifications library

#### DevOps & Architecture
- **Containerized database environment** using Docker and Docker Compose for MongoDB, enabling consistent development and deployment workflows

- **Structured application following MVC architecture** with clear separation of concerns: React components for View, Express routes/controllers for Controller, and Mongoose models for Model layer

- **Implemented centralized API client module** for frontend-backend communication with organized service functions for book and user operations
